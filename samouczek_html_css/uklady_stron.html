Układy stron w CSS

HTML CSS

Czym są układy stron?

Układ strony (layout) określa sposób rozmieszczenia elementów na stronie internetowej. Dzięki niemu możemy ustawić nagłówek, menu, treść oraz stopkę w odpowiednich miejscach.

CSS pozwala kontrolować pozycję i rozmiar elementów, dzięki czemu strona jest czytelna i estetyczna.

Układ z nagłówkiem, menu i treścią

Najprostszy układ strony składa się z kilku podstawowych sekcji. 

            <<>!DOCTYPE html>
            <html lang="pl">
            <head>
            <meta charset="UTF-8">
            <title>Page Title</title>

            <style>
            .header {
                background-color: lightblue;
                padding: 20px;
                text-align: center;
            }
            .menu {
                background-color: lightgray;
                padding: 10px;
            }
            .content {
                background-color: lightyellow;
                padding: 10px;
            }
            .footer {
                background-color: lightgray;
                padding: 10px;
                text-align: center;
            }
            </style>

            </head>

            <body>

            <div class="header">Nagłówek</div>
            <div class="menu">Menu</div>
            <div class="content">Treść strony</div>
            <div class="footer">Stopka</div>

            </body>
            </html>

Układ z Flexbox

Flexbox pozwala łatwo ustawiać elementy w rzędzie lub kolumnie. 

            <div class="container">
                <div class="menu">Menu</div>
                <div class="content">Treść</div>
            </div>
            
            .container {
                display: flex;
            }

            .menu {
                width: 25%;
                background-color: lightgray;
                padding: 10px;
            }

            .content {
                width: 75%;
                background-color: lightyellow;
                padding: 10px;
            }

Układ Grid

CSS Grid pozwala tworzyć układy wierszy i kolumn. 

            <div class="grid-container">
                <div class="grid-item">Menu</div>
                <div class="grid-item">Treść</div>
            </div>
            
            .grid-container {
                display: grid;
                grid-template-columns: 1fr 3fr;
                gap: 10px;
            }

            .grid-item {
                padding: 10px;
                background-color: lightblue;
            }

Ważne właściwości układów

  • display – grid / flex
  • width / height – rozmiary
  • padding / margin – odstępy
  • position – pozycjonowanie elementów
