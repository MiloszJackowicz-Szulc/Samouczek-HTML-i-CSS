HTML CSS

Co to jest model pudełkowy?

Model pudełkowy (Box Model) to sposób, w jaki przeglądarka traktuje każdy element HTML jako prostokątne pudełko.

Margin

Margin to zewnętrzny odstęp elementu od innych elementów. Jest przezroczysty i nie ma koloru tła.

<style>
  p{
    background-color: pink;
    margin: 40px;
  }
</style>

Border

Border to obramowanie elementu, które znajduje się między paddingiem a marginem.

<style>
  p{
    border: 3px solid black;
  }
</style>

Padding

Padding to odstęp wewnętrzny między treścią a obramowaniem. Powiększa “wnętrze” elementu.

<style>
  p{
    padding: 40px;
    background-color: pink;
  }
</style>

Skrócone zapisy

CSS pozwala skracać zapis margin i padding.

margin: 10px; → wszystkie strony

margin: 20px 40px; → góra/dół | lewo/prawo

margin: 10px 20px 40px; → góra | lewo/prawo | dół

margin: 10px 20px 30px 40px; → góra | prawo | dół | lewo

Ustawienia stron

Możemy ustawiać wartości dla konkretnej strony elementu.

margin-top
margin-bottom
margin-left
margin-right
