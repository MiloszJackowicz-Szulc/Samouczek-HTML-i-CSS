Pozycjonowanie w CSS

HTML CSS

Co to jest position?

W CSS właściwość position określa sposób pozycjonowania elementu na stronie.

Dzięki niej możemy kontrolować gdzie element się znajduje względem strony lub innych elementów.

Rodzaje position

  • static
  • relative
  • absolute
  • fixed
  • sticky

position: static

Domyślna wartość wszystkich elementów. Elementy układają się normalnie w dokumencie.

Static = brak specjalnego pozycjonowania

position: relative

Element przesuwa się względem swojej normalnej pozycji.

position: relative;
left: 20px;
BOX

position: absolute

Element pozycjonowany względem najbliższego rodzica z position: relative.

BOX
position: absolute;
top: 20px;
right: 20px;

position: fixed

Element przyklejony do ekranu – nie porusza się przy scrollowaniu(prawy dolny róg strony).

position: fixed;
bottom: 10px;
right: 10px;

position: sticky

Element zachowuje się początkowo jak position: relative, ale po przewinięciu strony przykleja się do określonego miejsca w oknie przeglądarki.

Sticky działa tylko wewnątrz swojego rodzica. Gdy dojdzie do końca kontenera, przestaje się przesuwać.

position: sticky;
top: 10px;
