Formularze w HTML

HTML CSS

Co to są formularze?

Formularze służą do pobierania danych od użytkownika. Możemy dzięki nim tworzyć logowanie, rejestrację, ankiety, formularze kontaktowe oraz wiele innych elementów interaktywnych.

Formularz tworzymy za pomocą znacznika <form>.

Tworzenie formularza

Podstawowy formularz składa się z pól do wpisywania danych oraz przycisku wysyłania.

<form>
  <input type="text">
  <input type="submit">
</form>

Najpopularniejsze typy input

  • text – zwykły tekst
  • password – hasło
  • email – adres e-mail
  • number – liczba
  • date – data
  • radio – pojedynczy wybór
  • checkbox – wiele wyborów
  • submit – wysyłanie formularza
