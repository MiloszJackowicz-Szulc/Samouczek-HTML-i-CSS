Co to jest HTML?

HTML (HyperText Markup Language) to hipertekstowy język znaczników służący do tworzenia stron internetowych. Nie jest językiem programowania, lecz językiem opisującym strukturę strony. Za pomocą HTML dodajemy nagłówki, tekst, obrazy, tabele i linki. Obecnym standardem jest HTML5.

HTML5 a starsze wersje

HTML5 wprowadził wiele usprawnień w porównaniu do HTML4 i XHTML. Dodano obsługę multimediów bez używania wtyczek oraz nowe znaczniki semantyczne, takie jak <header> i <nav>.

Przeglądarki internetowe

Przeglądarki interpretują kod HTML i wyświetlają go użytkownikowi jako stronę internetową. Pobierają dane z serwera i prezentują je w czytelnej formie.

Znaczniki HTML

Znaczniki określają strukturę strony internetowej. Mogą być kontenerowe (np. <p></p>) lub pojedyncze (np. <hr>). Wyróżniamy również elementy blokowe i liniowe.

DOCTYPE

DOCTYPE to deklaracja umieszczana na początku dokumentu, informująca przeglądarkę o wersji HTML użytej na stronie.

<!DOCTYPE html>

Podstawowa struktura dokumentu

<!DOCTYPE html>
<html>
<head>
    <title>Moja strona</title>
</head>
<body>
    Treść strony
</body>
</html>

Metadane strony

Metadane są niewidoczne dla użytkownika i znajdują się w sekcji <head>. Pomagają wyszukiwarkom zrozumieć zawartość strony.

<meta name="description" content="kurs html"> <meta name="author" content="Jan Kowalski"> <meta name="keywords" content="html, css">

Dodatkowe pojęcia

Parser sprawdza poprawność składni dokumentu. Witryna to zbiór powiązanych stron internetowych. Portal obejmuje wiele tematów, natomiast wortal skupia się na jednym zagadnieniu.

Programy do pisania HTML

Kod HTML można pisać w zwykłym notatniku, jednak najczęściej używa się Visual Studio Code lub Notepad++, ponieważ oferują wiele przydatnych funkcji.
