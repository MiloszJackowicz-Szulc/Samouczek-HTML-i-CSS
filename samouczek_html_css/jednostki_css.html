Jednostki w CSS

HTML CSS

Co to są jednostki w CSS?

Jednostki w CSS służą do określania rozmiarów elementów, takich jak szerokość, wysokość, marginesy czy rozmiary czcionek.

Zapisujemy je bez spacji pomiędzy liczbą i jednostką.

width: 300px;
font-size: 1.5rem;
margin: 10px;

Podział jednostek

Jednostki w CSS dzielimy na:

  • Jednostki bezwzględne
  • Jednostki względne

Jednostki bezwzględne

Jednostki bezwzględne mają stałą wielkość i nie zmieniają się w zależności od rozmiaru ekranu.

Nie są najlepszym rozwiązaniem przy tworzeniu stron responsywnych.

Jednostka Opis
px Najczęściej używana jednostka, około 1/96 cala.
cm Centymetry.
mm Milimetry.
in Cal (1in = 2,54cm = 96px).
pt Punkty (1pt = 1/72 cala).
pc Pica (1pc = 12pt).

Jednostki względne

Jednostki względne dopasowują swój rozmiar do innych elementów lub rozmiaru okna przeglądarki.

Dzięki nim łatwiej tworzyć strony responsywne.

em

Jednostka zależna od rozmiaru czcionki elementu nadrzędnego.

font-size: 2em;

ex

Rzadko używana jednostka zależna od wysokości znaków aktualnej czcionki.

ch

Odpowiada szerokości znaku „0” w aktualnej czcionce.

width: 40ch;

rem

Bazuje na rozmiarze czcionki elementu html.

Jest bardzo często wykorzystywana w nowoczesnych stronach.

font-size: 1.5rem;

vw

1vw oznacza 1% szerokości okna przeglądarki.

width: 50vw;

vh

1vh oznacza 1% wysokości okna przeglądarki.

height: 100vh;

vmin

1% mniejszego wymiaru okna przeglądarki (szerokości lub wysokości).

vmax

1% większego wymiaru okna przeglądarki (szerokości lub wysokości).

%

Wartość procentowa zależna od elementu nadrzędnego.

width: 50%;

Kiedy używać jakich jednostek?

  • px – gdy potrzebny jest dokładny rozmiar.
  • rem – do rozmiarów tekstu.
  • em – do elementów zależnych od tekstu.
  • % – do szerokości i wysokości względem rodzica.
  • vw i vh – do elementów zależnych od rozmiaru ekranu.
