figure i figcaption w HTML

HTML CSS

Co to jest figure?

<figure> to element HTML używany do grupowania treści graficznej (np. obrazów, wykresów, ilustracji) razem z podpisem.

Dzięki niemu możemy logicznie połączyć obraz i jego opis.

Co to jest figcaption?

<figcaption> to element, który dodaje podpis do elementu figure.

Zawsze umieszcza się go wewnątrz <figure>.

Podstawowa struktura

<figure>
  <img src="obraz.jpg" alt="Opis obrazu">
  <figcaption>To jest podpis obrazka</figcaption>
</figure>

Przykład działania

Obraz i podpis są traktowane jako jedna logiczna całość.

Dlaczego używamy figure?

  • Łączy obraz z podpisem
  • Poprawia semantykę strony
  • Ułatwia SEO
  • Pomaga czytnikom ekranowym

Gdzie stosować figure?

Według W3Schools element figure stosuje się do:

  • zdjęć
  • diagramów
  • wykresów
  • ilustracji edukacyjnych

Różnica: img vs figure

<img> → tylko obraz
<figure> → obraz + opis + kontekst
