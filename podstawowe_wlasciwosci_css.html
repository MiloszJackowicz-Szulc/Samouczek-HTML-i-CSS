Podstawowe właściwości CSS

HTML CSS

Color

Właściwość color służy do ustawiania koloru tekstu. Kolor można zapisać nazwą, kodem HEX lub RGB.

color: rgb(51, 51, 51);

Background-color

Właściwość background-color ustawia kolor tła elementu. Pozwala wyróżniać sekcje, przyciski oraz nagłówki.

background-color: lightblue;

Font-size

Font-size odpowiada za wielkość czcionki. Najczęściej używa się jednostki px.

font-size: 18px;

Font-family

Font-family określa krój pisma używany na stronie. Można podać kilka czcionek jako zapasowe.

font-family: Arial, sans-serif;

Font-weight

Font-weight ustawia grubość tekstu. Najczęściej używa się wartości bold lub normal.

font-weight: bold;

Font-style

Font-style służy do pochylania tekstu. Najczęściej stosowana wartość to italic.

font-style: italic;

Text-align

Text-align ustawia wyrównanie tekstu. Można wyrównywać do lewej, prawej, środka lub obu krawędzi.

text-align: justify;

Text-decoration

Text-decoration odpowiada za dekorację tekstu. Pozwala dodawać lub usuwać podkreślenia.

text-decoration: underline;

Text-transform

Text-transform zmienia wielkość liter bez edytowania HTML.

text-transform: uppercase;

Letter-spacing

Letter-spacing kontroluje odstępy pomiędzy literami.

letter-spacing: 2px;

Border-radius

Border-radius zaokrągla rogi elementów. Jest często używany przy przyciskach i kartach.

border-radius: 8px;

List-style-type

List-style-type zmienia wygląd punktów listy lub całkowicie je ukrywa.

list-style-type: none;

Text-shadow

Text-shadow dodaje cień do tekstu. Może poprawić czytelność napisów.

text-shadow: 2px 2px 4px black;

Box-shadow

Box-shadow dodaje cień do całego elementu. Pozwala uzyskać efekt głębi.

box-shadow: 0 4px 8px grey;

:hover

Pseudoklasa :hover działa po najechaniu kursorem na element.

p:hover {
    background-color: darkblue;
}

:active

Pseudoklasa :active działa podczas kliknięcia elementu.

a:active {
    color:red;
}

:visited

Pseudoklasa :visited stylizuje odwiedzone linki.

a:visited {
    color:purple;
}

:link

Pseudoklasa :link dotyczy linków, które nie zostały jeszcze odwiedzone.

a:link {
    color:#00FF00;
}

Poza wymienionymi tu wartościami istnieje też wiele innych. W ramach zobaczenia innych zachęcam odwiedzenie strony: W3Schools
