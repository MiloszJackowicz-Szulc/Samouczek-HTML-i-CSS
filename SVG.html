Co to jest SVG?

SVG (Scalable Vector Graphics) to format grafiki wektorowej wykorzystywany na stronach internetowych.

W przeciwieństwie do zwykłych obrazów JPG lub PNG, grafiki SVG nie tracą jakości podczas powiększania.

SVG jest zapisane w postaci kodu XML i może być umieszczane bezpośrednio w dokumencie HTML.

Znacznik SVG

Grafikę SVG tworzymy za pomocą znacznika <svg>.

<svg width="100" height="100">
    ...
</svg>

Rysowanie koła

Element <circle> służy do tworzenia okręgów i kół.

<svg width="120" height="120">
    <circle cx="60" cy="60"
    r="40" fill="skyblue" />
</svg>

Rysowanie prostokąta

Element <rect> służy do tworzenia prostokątów.

<svg width="150" height="100">
    <rect width="120"
    height="70"
    fill="orange" />
</svg>

Rysowanie linii

Element <line> pozwala tworzyć linie.

<svg width="200" height="100">
    <line x1="10" y1="50"
    x2="190" y2="50"
    stroke="red"
    stroke-width="5" />
</svg>

Rysowanie elipsy

Element <ellipse> tworzy elipsę.

<svg width="200" height="120">
    <ellipse cx="100"
    cy="60"
    rx="80"
    ry="40"
    fill="lime" />
</svg>

Zalety SVG

  • Nie traci jakości podczas powiększania.
  • Ma mały rozmiar pliku.
  • Można go edytować w kodzie HTML.
  • Wspiera animacje i style CSS.
  • Dobrze nadaje się do ikon i logo.
