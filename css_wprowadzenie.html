Co to jest CSS?

CSS (Kaskadowe Arkusze Stylów) jest językiem odpowiedzialnym za wygląd strony internetowej.

Dzięki niemu można zmieniać kolory, czcionki, rozmiary elementów oraz ich położenie na stronie.

CSS oddziela wygląd strony od jej struktury zapisanej w HTML, dzięki czemu tworzenie i zarządzanie stronami jest znacznie łatwiejsze.

To właśnie za pomocą CSS tworzy się również responsywność stron internetowych.

Historia CSS

Twórcą CSS był Håkon Wium Lie, który zaproponował ten język w 1994 roku.

Pierwsze standardy CSS1 i CSS2 pojawiły się pod koniec lat 90.

Później powstał CSS 2.1, który poprawił wiele błędów wcześniejszych wersji.

Największą zmianę przyniósł CSS3. Zamiast jednej wielkiej wersji wprowadzono niezależne moduły rozwijane osobno.

Dzięki temu pojawiły się nowoczesne rozwiązania takie jak: Flexbox, Grid, animacje i zaawansowane efekty wizualne.

Jak piszemy kod w CSS?

Reguła CSS składa się z:

  • selektora,
  • cechy (właściwości),
  • wartości.

Powyższa reguła ustawi niebieski kolor tekstu i wyśrodkuje go na stronie.

Co to jest kaskadowość?

Kaskadowość określa, który styl zostanie zastosowany, gdy jeden element posiada kilka różnych formatowań.

Im bliżej danego elementu znajduje się styl, tym większy ma priorytet.

Hierarchia stylów:

  1. Styl lokalny
  2. Rozciąganie stylu span
  3. Wydzielone bloki div
  4. Wewnętrzny arkusz stylów
  5. Zewnętrzny arkusz stylów
  6. Import arkusza stylów
  7. Domyślne ustawienia przeglądarki

1 oznacza najwyższy priorytet, natomiast 7 najniższy.

W czym piszemy CSS?

CSS zapisujemy w plikach posiadających rozszerzenie .css.

Najczęściej wykorzystuje się do tego Visual Studio Code lub Notepad++.

Zaleca się unikanie polskich znaków w nazwach plików CSS.

style.css

Jak podpiąć zewnętrzny arkusz CSS?

Aby połączyć dokument HTML z plikiem CSS, należy użyć znacznika link w sekcji head.

<link rel="stylesheet" href="style.css">

Dzięki temu jeden arkusz stylów może obsługiwać wiele podstron jednocześnie.

Przykład działania CSS

HTML:

<h1>Nauka CSS-a</h1>

<p style="color:purple;">
    Przykładowy tekst.
</p>

CSS:

body{
    background-color:#BBBBBB;
}

h1{
    text-align:right;
}

p{
    color:red;
}

Tekst w znaczniku p będzie fioletowy, ponieważ styl lokalny ma wyższy priorytet niż reguła z pliku CSS.
