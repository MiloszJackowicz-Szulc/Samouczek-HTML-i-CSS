Responsywność w CSS

HTML CSS

Co to jest responsywność?

Responsywność (Responsive Web Design) to technika tworzenia stron internetowych, dzięki której wyglądają one poprawnie na różnych urządzeniach.

Strona responsywna automatycznie dopasowuje układ elementów, rozmiary tekstu i obrazów do szerokości ekranu użytkownika.

Dzięki temu jedna strona może działać zarówno na komputerze, tablecie jak i telefonie.

Meta viewport

Aby strona poprawnie skalowała się na urządzeniach mobilnych, należy dodać odpowiedni znacznik meta.

<meta name="viewport"
content="width=device-width, initial-scale=1.0">

Dzięki temu szerokość strony zostanie dopasowana do szerokości urządzenia.

Jednostki względne

W responsywnych projektach często używa się jednostek procentowych (%) oraz jednostek viewportu (vw i vh).

width: 50%;
Widok strony:
Szerokość 50%

Media Queries

Media Queries pozwalają wykonywać różne style w zależności od szerokości ekranu.

@media(max-width:768px){

    .menu{
        flex-direction:column;
    }

}

Powyższy kod zostanie wykonany tylko wtedy, gdy szerokość ekranu będzie mniejsza lub równa 768px.

Responsywne obrazy

Obrazy powinny zmniejszać się razem z ekranem.

img{
    max-width:100%;
    height:auto;
}

Dzięki temu zdjęcie nigdy nie wyjdzie poza ekran urządzenia.

Responsywne układy

Do tworzenia nowoczesnych i elastycznych układów najczęściej wykorzystuje się Flexbox oraz CSS Grid.

Pozwalają one automatycznie rozmieszczać elementy w zależności od dostępnej przestrzeni.

Przykład Media Query

.box{
    width:100%;
}

@media(max-width:768px){

    .box{
        width:50%;
    }

}
Widok strony:

Na telefonach box będzie zajmował 50% szerokości.

Dlaczego responsywność jest ważna?

  • Strona działa na telefonach.
  • Strona działa na tabletach.
  • Strona działa na komputerach.
  • Poprawia wygodę użytkownika.
  • Jest standardem nowoczesnych stron internetowych.
