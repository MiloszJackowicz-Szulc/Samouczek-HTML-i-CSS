Czym jest background w CSS?

Właściwości background w CSS służą do ustawiania tła elementów. Możemy zmieniać kolor, obraz oraz jego zachowanie.

Najczęściej używane właściwości:

  • background-color
  • background-image
  • background-repeat
  • background-position
  • background-attachment
  • background (skrót)

background-color

Ustawia kolor tła elementu.

body {
  background-color: lightblue;
}

Widok:
Tło kolorowe

background-image

Ustawia obraz jako tło elementu.

body {
  background-image: url("tlo.jpg");
}

Domyślnie obraz powtarza się i wypełnia cały element.

background-repeat

Określa sposób powtarzania obrazu tła.

  • repeat
  • repeat-x
  • repeat-y
  • no-repeat
body {
  background-image: url("tlo.jpg");
  background-repeat: no-repeat;
}

background-position

Ustawia pozycję obrazu tła.

body {
  background-image: url("tlo.jpg");
  background-position: center;
}

background-attachment

Określa czy tło przewija się razem ze stroną.

body {
  background-image: url("tlo.jpg");
  background-attachment: fixed;
}

fixed oznacza, że tło pozostaje w miejscu podczas scrollowania.

background (skrót)

Jedna właściwość może zastąpić kilka innych.

body {
  background: lightblue url("tlo.jpg") no-repeat center;
}

W jednej linijce ustawiamy kolor, obraz, powtarzanie i pozycję.
