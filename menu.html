Menu w HTML

HTML CSS

Co to jest menu w HTML?

Menu w HTML to zestaw linków, które pozwalają użytkownikowi poruszać się po stronie internetowej.

Najczęściej buduje się je za pomocą znaczników: <nav>, <ul>, <li> oraz <a>.

Podstawowa struktura menu

Według W3Schools najprostsze menu wygląda tak:

<nav>
  <ul>
    <li><a href="#">Strona główna</a></li>
    <li><a href="#">O nas</a></li>
    <li><a href="#">Kontakt</a></li>
  </ul>
</nav>

Menu poziome (horizontal menu)

Menu poziome to najczęściej używany typ menu na stronach internetowych.

nav ul li {
  display: inline;
}
Linki ustawione obok siebie tworzą pasek nawigacyjny.

Przykład menu poziomego

Home About Services Contact
.menu a {
  color: white;
  text-decoration: none;
  margin-right: 15px;
}

Menu pionowe (vertical menu)

Menu pionowe często stosuje się w panelach bocznych strony.

ul {
  list-style-type: none;
  padding: 0;
}

li a {
  display: block;
}

Przykład menu pionowego

  • Strona główna
  • O nas
  • Oferta
  • Kontakt

Najważniejsze właściwości menu

  • list-style-type – usuwa kropki listy
  • display: inline – ustawia elementy obok siebie
  • display: block – robi pełne klikalne pola
  • text-decoration – usuwa podkreślenie linków
  • :hover – efekt po najechaniu
