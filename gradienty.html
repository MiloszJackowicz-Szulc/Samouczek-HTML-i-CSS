HTML CSS

Co to są gradienty?

Gradienty pozwalają tworzyć płynne przejścia pomiędzy kolorami bez używania obrazków. W CSS najczęściej wykorzystuje się gradienty liniowe (linear-gradient) oraz promieniste (radial-gradient).

Dzięki gradientom można tworzyć nowoczesne tła, przyciski oraz efektowne sekcje strony internetowej.

Gradient liniowy (linear-gradient)

Gradient liniowy tworzy przejście kolorów w określonym kierunku.

background: linear-gradient(to right, #ff512f, #dd2476);

Gradient pionowy

Kierunek przejścia można ustawić również od góry do dołu.

background: linear-gradient(to bottom, #ff512f, #dd2476);

Gradient pod kątem

Zamiast kierunku można podać kąt nachylenia gradientu.

background: linear-gradient(45deg, #ff512f, #dd2476);

Gradient promienisty (radial-gradient)

Gradient promienisty rozchodzi się od środka elementu na zewnątrz.

background: radial-gradient(circle, #ff512f, #dd2476);

Gradient z wieloma kolorami

Gradient może zawierać więcej niż dwa kolory.
