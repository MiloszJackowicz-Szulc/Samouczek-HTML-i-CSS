Co to są animacje?

Animacje CSS pozwalają zmieniać wygląd elementów bez używania JavaScript. Możemy przesuwać elementy, zmieniać kolory lub obracać obiekty.

Właściwość animation

Właściwość animation uruchamia animację przypisaną do elementu.

animation: nazwa 3s infinite;

Tworzenie animacji

Animacje tworzymy przy pomocy @keyframes.

@keyframes nazwa{
0%{ background:red; }
100%{ background:blue; }
}
