Selektory w CSS

HTML CSS

Co to są selektory w CSS?

Selektory to wzorce, które pozwalają wskazać konkretne elementy HTML, które mają zostać ostylowane.

Dzięki nim możemy zmieniać wygląd tylko wybranych części strony bez wpływania na całą strukturę.

Selektor uniwersalny *

Selektor * działa na wszystkie elementy strony.

* { margin: 0; padding: 0; }

Selektor typu (E)

Wybiera wszystkie elementy danego typu, np. p lub h1.

p { color: red; }

Selektor potomka (E F)

Styluje wszystkie elementy znajdujące się wewnątrz innego elementu.

div p { color: blue; }

Selektor dziecka (E > F)

Działa tylko na bezpośrednie dzieci elementu.

div > p { color: green; }

Selektor sąsiada (E + F)

Styluje element znajdujący się bezpośrednio po innym.

h1 + p { color: orange; }

Selektor ogólnych braci (E ~ F)

Wybiera wszystkie elementy tego samego poziomu po elemencie E.

h1 ~ p { color: purple; }

Selektor klasy

Klasa pozwala stylować wiele elementów jednocześnie.

.nazwa-klasy { color: red; }
<div class="nazwa-klasy">Tekst</div>

Selektor ID

ID służy do stylowania jednego, unikalnego elementu.

#identyfikator { color: blue; }
<div id="identyfikator">Tekst</div>
